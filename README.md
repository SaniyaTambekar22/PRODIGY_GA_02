# PRODIGY_GA_02
Image generation with Pre-trained Models
🚀 Excited to share my latest Generative AI project — Image Generation with Stable Diffusion!
As part of my internship Prodigy InfoTech, I successfully built a text-to-image generation system using the Stable Diffusion model. Here's a quick breakdown:
🔍 Problem Statement : Creating realistic and visually coherent images from simple text descriptions is one of the most exciting challenges in Generative AI. 
This project explores how pre-trained diffusion models can generate stunning images purely from natural language prompts.
🛠️ Tech Stack :
🐍 Python
🤗 Hugging Face Diffusers
🎨 Stable Diffusion v1.5 (RunwayML)
🔥 PyTorch
📊 Google Colab (T4 GPU)
🖼️ PIL / IPython Display
⚙️ Workflow & Results :
📝 Designed creative text prompts as image descriptions
🔧 Loaded pre-trained Stable Diffusion pipeline with float16 precision for GPU efficiency
🎨 Generated high quality images purely from text prompts
✅ Produced photorealistic outputs like sunsets, landscapes and animals
✅ Experimented with multiple prompt styles — photorealistic, cartoon, cyberpunk
✅ Observed how prompt wording directly impacts image quality and accuracy
✅ Successfully demonstrated text-to-image translation using state-of-the-art GenAI
